# meerkatplace

Personal Claude Code plugin marketplace.

## Install

Add this marketplace to Claude Code:

```
/plugin marketplace add <your-github-username>/meerkatplace
```

Then install plugins from it:

```
/plugin install worklog
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [worklog](./worklog) | Log what you accomplished in a Claude Code session to `~/worklog/worklog.md` |
